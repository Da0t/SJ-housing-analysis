# San Jose Housing Affordability Analysis

## Overview
This project analyzes housing affordability in San Jose by examining long-term
trends in median home prices, rental costs, and household income. The goal is to
understand whether housing costs have outpaced income growth and to explore what
current trends suggest about future affordability.

## Research Question
How has housing affordability in San Jose changed over time relative to household
income, and what do recent trends suggest about future affordability?

## Key Metrics
- Home Price-to-Income Ratio
- Rent Burden (Annual Rent / Household Income)

## Data Sources
- Zillow: housing prices and rental data
- U.S. Census Bureau: median household income
- Bureau of Labor Statistics: Consumer Price Index (CPI) for inflation adjustment

## Tools & Methods
- Python (pandas, numpy, matplotlib, seaborn)
- Exploratory Data Analysis (EDA)
- Regression-based machine learning models for forecasting affordability
